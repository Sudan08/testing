# Routine Management System of Herald College Kathmandu

## Requirements

- nodejs >v16
- npm or yarn

## Project Setup

Skip this part if you already have `yarn` installed

### Installing Yarn

```bash

npm install  yarn --location=global

```

### Installing dependencies

```bash

yarn

```

### Starting Server

```bash

yarn dev

```
